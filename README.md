# liveplayer-vc

LivePlayer H5 播放器， 使用 vue-cli 集成示例

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
